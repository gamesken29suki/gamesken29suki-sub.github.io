# Gamesken29suki - 個人サイト

このリポジトリは個人サイトの静的ファイルです。Bootstrap と GSAP を CDN で読み込み、簡易的な自己紹介、リンク、ポートフォリオ、ブログ、お問い合わせフォームを含みます。

必須： `index.html`, `styles.css`, `script.js` を編集して自分の情報に置き換えてください。

デプロイ（GitHub Pages）

1. リポジトリを GitHub にプッシュします。
2. リポジトリの Settings → Pages で `main`（または `gh-pages`）ブランチを選び、ルートを公開先に設定します。
3. 数分待つと https://<ユーザー名>.github.io/<リポジトリ名>/ で公開されます。

Google Analytics の設定

- `index.html` の gtag 初期化内の `G-XXXXXXXXXX` を実際の計測ID に置き換えてください。

その他

- `images/` に作品画像を置くとポートフォリオのカードに表示できます。
- 追加でホストしたい場合は `CNAME` を使ってカスタムドメインを設定してください。
